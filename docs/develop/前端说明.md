# 前端开发说明

## 目录结构

```
|-- frontend
    |-- public          # 静态资源文件夹
    |-- src             # 源文件
        |-- assets
        |-- router
        |-- utils
        |-- views
        |-- App.vue
        |-- main.js
    |-- wailsjs         # 后端自动生成的接口文档(请勿修改)
    |-- README.md       # 说明文档
    |-- index.html
    |-- package.json    # vue项目描述文件
    |-- vite.config.js  # vite配置文件
```
